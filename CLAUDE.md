# Dune Hackathon — ESG Value Creation Calculator

## Project context
Dune Partners is een ESG-adviesbureau. Dit project is een hackathon-MVP voor een **ESG Value Creation Calculator**: een tool die de financiële waarde van ESG-verbeteringen (verzuim, verloop, etc.) tastbaar maakt voor klanten.

## Werkdirectory structuur
```
docs/
  use-case-1-brainstorm.md   # Open vragen + beslissingen voor Use Case 1
  aannames-en-benchmarks.md  # Kosten- en benchmarkdata zodra bekend
  demo-casus.md              # Voorbeeldbedrijf / demo-scenario
src/                         # Code (nog leeg, we bouwen nog niks)
```

## Fase
We zitten in de **brainstormfase**. Nog geen code — eerst requirements en aannames vastleggen.

## Use cases in scope (hackathon)
1. **ESG Value Creation Calculator** — verzuim- en verloopreductie naar euro's
2. (Roadmap) Aanvullende KPI's: ongevallenratio, energiereductie, elektrificatie wagenpark, datalek-risicokosten
